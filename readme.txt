== Description ==
After the frustration with the existing popup plugins I decided to create a new WordPress
Popup plugin from scratch named <strong>WP Popup Plugin</strong> 
and after months of development and hard work the first version is 
ready Please download the plugin and use it on your site and let me know your 
<a href="https://wordpress.org/support/view/plugin-reviews/unlimited-popups?filter=5">feedback</a>. I hope you'll also love my plugin like I do.
Recoded from scratch. This is a totally new plugin. It's compatible with the major form plugins like
* Contact form 7
* Gravity Forms
* Ninja Forms
* Mailchimp for WP
* Slider plugins
* Any generic html, images, generic form, iframe, video.

== Installation ==

1. Unzip and Upload the directory 'unlimited-popups' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Popups menu and add as many popups as needed
